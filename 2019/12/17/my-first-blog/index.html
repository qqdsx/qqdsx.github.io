hexo&next主题安装过程中遇到的问题

  • _config.yml的deploy如何写 
deploy:
  type: git
  repository: https://github.com/qqdsx/qqdsx.github.io
theme: next
  • hexo使用git提交报错
    npm install hexo-deployer-git --save
  • 安装next主题后，输入命令hexo d我的主页github.io并没显示next的主题。
    hexo clean 后删除.gitignore文件再hexo d
  • 在机器上有git并且与github配置好后，安装过程中使用到的命令
    按顺序依次是：
    安装hexonpm install -g hexo
    mkdir ~/hexo，进入~/hexo初始化hexohexo init
    安装Next主题git clone https://github.com/iissnan/hexo-theme-next themes/next
    验证是否成功，本地hexo g+hexo s,验证成功后上传githubhexo d

hexo下markdown标题不显示

一般是因为使用时的语法不严格，比如错误事例为##标题。正确语法是## 标题。（有点坑）

使用Next的主题时，URL末尾会带有%20

%20是空格，需要修改 your_hexo/themes/next/_config.yml文件的第95行
home: / || home 改为home: /|| home
archives: / || archives 改为 archives: /|| archives（真坑）
